How to make this brief live on the site

This page renders in your browser only. To publish the brief on the home gallery, commit the JSON to briefs/data/ and run the build:

node briefs/build.mjs
git add briefs/data briefs/manifest.json "<Client> Mission Brief.html"
git commit -m "Add brief for <Client>"
